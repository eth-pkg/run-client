## Running a Node

<details>
  <summary>besu - lighthouse</summary>

  Open three terminals and execute the following commands:

  Terminal 1:
  ```bash 
  bash run-client.sh --network testnet \
                     --consensus-client lighthouse \
                     --execution-client besu \
                     --run execution \
                     --with-validator 
  ```

  Terminal 2:
  ```bash
  bash run-client.sh --network testnet \
                     --consensus-client lighthouse \
                     --execution-client besu \
                     --run consensus \
                     --with-validator
  ```

  Terminal 3:
  ```bash
  bash run-client.sh --network testnet \
                     --consensus-client lighthouse \
                     --execution-client besu \
                     --run validator \
                     --with-validator 
  ```

</details>
